桌面蟑螂 🪳

Desktop Cockroach — 你的桌面赛博小宠物

🪟 Windows 🍎 Mac · Intel 芯片 🍎 Mac · Apple 芯片

它能干什么

🕳️

点墙洞钻蟑螂

桌面常驻一个逼真墙洞，点一下钻出一只，可反复点、可拖到任意位置（还能跨屏拖）。

🐜

会爬 · 会停 · 会飞

随机乱爬、平滑转向；偶尔停下发呆转头；小概率张开翅膀煽动着飞一段。

🖥️

多屏乱窜

透明覆盖层铺满所有显示器，蟑螂能从一块屏爬/飞到另一块屏。

🪰

苍蝇拍拍死

鼠标移到蟑螂上光标变苍蝇拍，抬起蓄势、点击向下拍，配打击特效。

⏱️

产出方式可选

菜单栏可选：点击产出 / 定时自动产出（间隔可调）/ 随机自动产出。

🫥

不打扰

平时鼠标穿透，不影响你正常使用其他软件；托盘一键清空 / 退出。

支持 macOS（Apple Silicon / Intel）与 Windows。安装包未做代码签名，首次打开时 macOS 请右键「打开」，Windows 请点「更多信息 → 仍要运行」；Windows 若安装受阻可用免安装 zip 版。
