# 桌面蟑螂 Desktop Cockroach — 设计文档

日期：2026-07-14
状态：已确认，待生成实现计划

## 一句话

一个 macOS 后台应用：桌面上常驻一个墙洞，点它就从洞里钻出一只自带爬行动效的蟑螂，蟑螂沿随机路径满屏乱窜，跑出屏幕边缘自动销毁，被点中则播放打击特效后渐隐消失。

## 形态与平台

- 平台：macOS。技术栈 Electron（透明 + 置顶 + 鼠标穿透是刚需，Web 做不到，原生 Swift 成本过高）。
- 后台应用，**没有主窗口**。一个透明、全屏、始终置顶（`alwaysOnTop`，浮在普通窗口之上）的覆盖层铺满屏幕。
- 菜单栏托盘图标，右键菜单：`清空所有蟑螂` / `退出`。这是唯一的控制入口。

## 鼠标穿透（核心机制）

- 覆盖层默认整体穿透：`setIgnoreMouseEvents(true, { forward: true })`。用户点击/操作屏幕上其他软件完全不受影响。
- `forward: true` 让主进程/渲染进程仍能收到 `mousemove`，据此实时判断光标是否悬停在**墙洞**或**某只蟑螂**的命中区域上。
- 悬停在可点元素上时 → `setIgnoreMouseEvents(false)` 临时接管点击；光标移开 → 恢复穿透。
- 命中判定用元素的几何包围盒即可（蟑螂用其当前包围矩形，墙洞用洞口圆/矩形），无需逐像素。

## 墙洞

- 常驻，固定显示，**默认位置：屏幕右下角**。
- **可拖动**：按住墙洞可拖到屏幕任意位置（拖动期间接管鼠标）。
- 点击墙洞 → 钻出一只蟑螂，配一段"从洞口爬出来"的短动画（从洞口位置淡入 + 向外爬出）。
- 可反复点击，多只并存。
- **数量上限：同时最多 30 只**。达到上限后点击墙洞不再新增（可给个轻微抖动反馈表示满了）。

## 蟑螂

- 外观：SVG/CSS 绘制，**自带动效**——6 条腿交替摆动、触角轻抖、身体朝向随移动方向转向（`transform: rotate`）。
- 移动：每只蟑螂是独立对象，拥有自己的随机路径状态：
  - 随机初速度与方向，走一段后偶尔随机拐弯 / 变速，营造"乱窜"感。
  - 由全局单个 `requestAnimationFrame` 循环统一驱动所有蟑螂的位置与动画帧更新。
- **销毁：** 蟑螂完全移出屏幕边缘 → 从场景与对象列表中移除。
- **拍死（点中）：** 命中瞬间播放打击特效——冲击波扩散圈 + 碎点飞溅 + 蟑螂压扁翻肚，随后整体渐隐消失，动画结束后移除对象。

## 技术结构

- `src/main/index.ts` —— Electron 主进程：创建透明置顶全屏窗、管理鼠标穿透切换、创建托盘。
- `src/main/tray.ts` —— 托盘图标与菜单（清空 / 退出），通过 IPC 通知渲染进程。
- `src/preload/` —— 暴露安全的 IPC 通道（如"清空所有蟑螂"、命中区上报以切换穿透）。
- `src/renderer/` —— 页面渲染：
  - `hole`：墙洞元素（渲染、点击、拖动）。
  - `cockroach`：单只蟑螂对象（SVG 结构 + 自身移动/朝向/腿部动画状态 + 生命周期）。
  - `stage`：管理所有蟑螂列表、统一 rAF 驱动、上限控制、命中检测、销毁回收。
  - `splat`：拍死打击特效（冲击波 + 碎点 + 压扁 + 渐隐）。
- 命中区/穿透切换：渲染进程根据 `mousemove` 判断光标是否在墙洞或任一蟑螂包围盒内，经 IPC 请求主进程切换 `setIgnoreMouseEvents`。

## 模块边界（各单元职责）

| 单元 | 做什么 | 依赖 |
|---|---|---|
| main window | 建透明置顶窗、穿透开关 | Electron |
| tray | 菜单入口，发 IPC 指令 | Electron |
| hole | 画墙洞、响应点击/拖动、发"生成蟑螂"事件 | DOM |
| cockroach | 单只蟑螂的外观 + 自身运动/动画/生命周期 | 纯逻辑，可单测运动学 |
| stage | 蟑螂集合管理、rAF 驱动、上限、命中、回收 | cockroach |
| splat | 拍死特效 | DOM/CSS |

蟑螂的运动学（下一步位置、朝向、是否出界）应为可独立测试的纯函数。

## 明确不做（YAGNI）

- 不做 Windows/Linux（先只 macOS）。
- 不做多种蟑螂皮肤、不做设置面板、不做音效（可作为后续增强，非首版）。
- 不做逐像素命中（包围盒足够）。
- 不做视频/图片素材、AI 生成（纯代码绘制）。

## 成功标准

1. 启动后无主窗口，右下角出现墙洞，托盘有图标。
2. 平时鼠标穿透，不影响正常使用其他软件。
3. 点墙洞钻出带动效的蟑螂，可反复点、多只并存、上限 30。
4. 蟑螂随机乱爬、朝向跟随移动方向，出屏销毁。
5. 点中蟑螂播放打击特效并渐隐消失。
6. 托盘可清空所有蟑螂、可退出应用。
